Setup Required

After enabling scheduled sync, you need to set up one of the following:

  • Cron Job: */{{ settings.sync_interval_minutes|default:"60" }} * * * * cd /opt/netbox/netbox && /opt/netbox/venv/bin/python manage.py schedule_meraki_sync
  • Systemd Timer: Install the provided systemd service files
  • Continuous Service: python manage.py schedule_meraki_sync --continuous

See deployment documentation for complete setup instructions.
