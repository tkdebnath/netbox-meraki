## How to Capture Screenshots

1. Log in to your NetBox instance
2. Navigate to Plugins > Meraki Sync
3. Capture screenshots using your browser's screenshot tool or Snipping Tool
4. Save screenshots with the exact filenames listed above
5. Ensure screenshots are clear and at appropriate resolution (1280x720 or higher recommended)
6. Remove any sensitive information (API keys, internal IPs) before committing
